# Afrog 从入门到入土系列教程大纲

本系列教程旨在帮助安全研究人员、红队工程师以及漏洞挖掘爱好者，从零开始掌握 Afrog 漏洞扫描器的使用、配置及高级 PoC 编写技巧。

## 📅 连载计划

### 第一期：初识 Afrog 与 Hello World
- **目标**：完成安装并进行第一次扫描。
- **内容**：
  - Afrog 是什么？（核心优势：快、准、稳）
  - 保姆级安装教程（Windows/Mac/Linux 二进制 & 源码）
  - 你的第一次扫描：`afrog -t example.com`
  - 解读扫描报告（HTML 报告的秘密）

### 第二期：不仅是扫描 - 命令行参数全解
- **目标**：掌握 CLI 的各种姿势，实现精准打击。
- **内容**：
  - 目标指定：单目标、文件批量、端口扫描联动 (`-ps`)
  - 过滤的艺术：按关键词搜索 PoC (`-s`)、按等级过滤 (`-S`)
  - 排除法：如何跳过不想扫的漏洞 (`-ep`)
  - 输出管理：JSON 与 HTML 的抉择

### 第三期：武装到牙齿 - 配置文件深度解析
- **目标**：配置反连平台（OOB）与通知系统。
- **内容**：
  - `afrog-config.yaml` 在哪里？
  - 配置反连平台（Ceye, Dnslog, Revsuit）：让盲注无处遁形
  - 钉钉/企业微信群机器人告警：躺着也能收漏洞
  - 空间测绘联动（ZoomEye）：自动寻找目标

### 第四期：炼金术士 - PoC 编写入门
- **目标**：看懂并能修改简单的 PoC。
- **内容**：
  - PoC 的解剖学：ID、Info、Rules
  - 核心逻辑：Request（请求）与 Expression（表达式）
  - 实战演练：编写一个检测 Git 信息泄露的 PoC
  - 本地调试技巧：`-P` 参数的妙用

### 第五期：大魔法师 - 高级 PoC 技巧
- **目标**：编写依赖指纹、反连检测等复杂逻辑的 PoC。
- **内容**：
  - 变量 (`set`) 与 内置函数（随机数、编码）
  - 指纹门控 (`requires`)：为什么你的 PoC 不跑？
  - 反连检测：`oob()` 与 `oobCheck()` 的实战
  - 逻辑组合：多步骤攻击链的实现

### 第六期：开源精神 - 如何提交高质量 PoC
- **目标**：参与社区贡献，成为 Contributor。
- **内容**：
  - 避免误报的黄金法则
  - 什么是“高质量”的 PoC？
  - GitHub 提交 PR 的流程
  - 常见被拒原因分析


---
*注：本大纲内容可能根据 Afrog 版本迭代进行微调。*
